About SSH Folder Sync

Technical specification and information about the plug-in.

This plugin implements a secure folder synchronizer that retrieves remote directories from SSH target nodes to the local file system. It provides real-time progress streaming, differential checks, and handles folder synchronizations efficiently.

Technical Features:

  • Asynchronous Threading: Employs non-blocking background runners. PHP delegates execution to an isolated Python script which monitors transfer natively.
  • rsync Engine Optimization: Leverages the fast incremental rsync copy tool, compressing files over the SSH tunnel, preserving absolute permissions, and resuming failed downloads.
  • aaPanel Host Integration: Scans `/www/server/panel/data` SQLite files recursively and handles password security decryption internally.

Compatibility requirements:

Designed for Linux distributions supported by aaPanel (Ubuntu, CentOS, Debian). Standard `rsync` must be installed on both local and target remote hosts (standard in most OS configurations).
